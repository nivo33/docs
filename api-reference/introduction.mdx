Welcome to the API documentation for the Orchestra API. This API provides direct access to the Orchestra research methods
and capabilities, allowing you to conduct deep research and enrichments on both companies and people.


<Info>
  **Prerequisite**: You need an API key to access this API. To obtain one, please contact support.
</Info>

## Authentication

All API endpoints are authenticated using an API key, set as an "x-api-key" header.

```json
{ "x-api-key": "YOUR_API_KEY" }
```

## Next Steps

To get started with the API, view the relevant endpoint in the sidebar. Each endpoint has a description, request, and response examples.
