## API Key

All API endpoints are authenticated using an API key, set as an "x-api-key" header. The API key can be obtained by contacting support.

```json
{ "x-api-key": "YOUR_API_KEY" }
```

## Getting Started

To get started with the API, view the relevant endpoint in the sidebar. Each endpoint has a description, request, and response examples.
